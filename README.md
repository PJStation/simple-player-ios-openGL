最简iOS播放器使用ffmpeg软解码视频数据，SDL_audio播放音频数据，SDL_thread管理线程，openGL渲染YUV420数据到UIView。该项目仍然是基础项目，适合学习播放流程、解码流程、线程管理、数据队列等，不适合正式使用到项目中。

项目中缺少的FFmpeg-iOS文件，请自行编译添加，添加后记得配置好路径。
